{% endhint %}



## HTML5에서 추가된 기능

* [ ] 멀티미디어(video, audio) 지원
* [ ] 2차원 그래픽(svg, canvas), 3차원 그래픽(CSS3, WebGL) 지원
* [ ] 서버와의 소켓 통신을 지원(양방향 통신)
* [ ] 하드웨어(camera, sensor) 제어 지원
* [ ] 오프라인 상태에서 동작 지원
* [ ] 시멘틱 태그
