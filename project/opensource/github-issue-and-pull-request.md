## issue & PR 템플릿이 무엇인가?

기여자가 issue, PR을 할 때 포함시키도록 하고 싶은 정보를 사용자 지정하고 표준화하는 마크다운 문서



## 코드 샘플

> ISSUE\_TEMPLATE.md, PULL\_REQUEST\_TEMPALTE.md 마크다운 문서로 저장해야 한다.

```markdown
<!-- ISSUE_TEMPLATE.md ->
# Detail

# TODO

# ETC
```



```markdown
<!-- PULL_REQUEST_TEMPLATE.md ->
# issue

# TODO

# ETC
```



## 템플릿 저장 위치

> 위치는 아래 3가지 이용할 수 있다.

* `/` : root에 저장
* `docs/` : 디렉터리에 저장
* `.github/`: 숨김 디렉터리에 저장
